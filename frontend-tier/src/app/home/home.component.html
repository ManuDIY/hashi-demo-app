Multi-tier Application Demo
This application is designed to use Consul Connect in a Kubernetes environment to demonstrate service mesh functionaltiy. Intentions are created within Consul to dictate whether communication is "Allowed" or "Denied". Application communication occurs through sidecar proxies on each container pod.
The application itself is a 3-Tier application consisting of an Angular frontend running behind Nginx, a Python (Flask) based API, and a PostgreSQL database. The UI framework is Clarity UI. You can see the code for this project on GitHub.
