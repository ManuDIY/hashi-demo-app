Multi-tier Application Demo
This application is designed to demonstrate Consul functionality. Consul provides consistent network across clouds, both public and private. It provides solutions for Service Discovery, Health Checking, Service Network Automation and enables a service mesh for multiple runtime platforms. This application can be used to can demonstrate capabilities like Intentions, L7 Routing, and Observability.
The application itself is a 3-Tier application consisting of an Angular frontend running behind Nginx, a Python (Flask) based API, and a PostgreSQL database. The UI framework is ClarityUI. You can see the code for this project on GitHub.
